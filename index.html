Google Drive Storage
Connected to Google Drive

New courses you upload will be saved to your CourseHub folder on Google Drive. Your laptop stores zero video data.

Connect Google Drive so videos are stored there instead of your laptop. Follow these one-time steps:

1
Run via local server
Open a terminal in the app folder and run:
python -m http.server 8080
Then open http://localhost:8080 in your browser.
2
Create a Google Cloud OAuth Client
console.cloud.google.com → Credentials
→ Create → OAuth 2.0 Client ID → Web app
→ Add http://localhost:8080 as Authorized JS origin
→ Enable Google Drive API in the library
3
Paste your Client ID below
Copy the Client ID from Google Cloud (ends in .apps.googleusercontent.com)
